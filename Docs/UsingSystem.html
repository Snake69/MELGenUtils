Notes for using MEL Genealogy Utilities (MELGenUtils).

MELGenUtils is menu based. The Main (top level) Menu consists of items:  "Import, Create or Remove a Family DataBase", "Verify Family Data",
"Family DataBase Activation", "Display Family Data", "Edit Family Data", "Reports", "Search/Inspect/Analyze Family Data",
"Miscellaneous Functions" and "Help/Info/Docs".

The following is a listing of all items in the top level menu (Main Menu) and all sub-menus. Click on an item to see it's description.
Note that MELGenUtils is a work-in-progress and not all menu items (functions) are currently usable. Also note that not all menu items are
available depending upon whether or not the node/server is running, and whether or not data is available.

Note - Console error messages regarding "SSE" (Server Side Events) may occur when displaying (or re-displaying) the top level menu (Main Menu).
These are transient error messages and should not have any effect upon processing.
