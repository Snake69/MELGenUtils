Create HTML Version of Family DataBase

This function will create a HTML version of the active Family DataBase. HTML makes it easier to jump around
in the data when viewing on a computer. (The Family DataBase must first be successfully verified.) The HTML
version of the active Family DataBase is suitable to be used as a Web page. (See "Help/Info/Docs -> Using MELGenUtils" and
then click on "Create HTML Version of Family DataBase" for an explanation of how to do this.)

Click the 'Continue' button to proceed.


Creation of a HTML version may take some time depending upon the size of the DataBase.
